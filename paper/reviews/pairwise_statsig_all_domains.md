### Proposed (all domains)

> Statistical Reliability. For all three domains we conducted 2 independent runs per condition and test statistical significance using paired permutation tests (100k permutations, paired by task ID, Holm-Bonferroni corrected; Table XX). Pooling across domains, both Text → Realistic and Clean → Realistic gaps are statistically significant for all three voice providers (all p ≤ 0.002), as is the reasoning Text → Clean gap (all p < 0.001). Combined voice providers achieve 31–52% (Clean) and 24–36% (Realistic), well below text baselines of 64% (non-reasoning) and 85% (reasoning). The non-reasoning Text → Clean gap is significant for Gemini (p < 0.001) and OpenAI (p = 0.001) but not for xAI (p = 0.231), driven by Telecom where xAI's Clean rate (57.5%) significantly *exceeds* the non-reasoning text baseline by +20.6 pp. This anomaly disappears for Realistic conditions: xAI's Telecom Realistic rate (37.7%) is comparable to the text baseline, and the combined non-reasoning Text → Realistic gap is significant for all providers (all p < 0.001).
>
> Per-domain results (Tables XX–XX) show that significance patterns vary with domain complexity and task count. In Retail (n = 114), all comparisons — including both Text → Clean and Text → Realistic — are significant for all providers (all p < 0.05), with even the narrowest gap reaching significance (p = 0.032). In Airline (n = 50), the non-reasoning Text → Clean gap is significant only for Gemini (p < 0.001); OpenAI's Clean rate of 51% nearly matches the text baseline. The non-reasoning Text → Realistic gap is wider: significant for Gemini (p = 0.002) and xAI (p = 0.009) but marginal for OpenAI (p = 0.066). The Clean → Realistic gap is significant only for xAI (p = 0.015). In Telecom (n = 114), the Clean → Realistic gap is significant only for xAI (p < 0.001); Gemini and OpenAI show marginal but non-significant declines (p = 0.062 and p = 0.059). The non-reasoning Text → Realistic gap is non-significant for xAI (p = 0.916), reflecting that xAI's Realistic rate (37.7%) matches the non-reasoning text baseline (36.8%).
