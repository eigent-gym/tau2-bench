# Synthetic Banking Knowledge Domain

This shared domain contains reusable environment artifacts generated by the
`dategen_tau3_banking` pipeline:

- 91 synthetic KB documents covering eight banking scenario modules;
- one versioned base transactional database;
- a machine-readable registry for the initial business and user tools.

The fictional product brand used inside the current KB is **Northstar Bank**.
The technical domain name is intentionally brand-neutral so future world
versions can replace the brand without changing Harbor or tau2 integration.

Per-task user scenarios, DB overlays, required-document IDs, and reference
actions do not belong here. Harbor task exporters supply those through the
task configuration at runtime.
